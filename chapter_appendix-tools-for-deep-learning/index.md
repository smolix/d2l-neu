# Tools for Deep Learning
:label:`chap_appendix_tools`


This chapter connects the book's executable notebooks to the systems around
them: interactive editors, hosted runtimes, rented accelerators, personal
hardware, model artifacts, distributed training, and model serving. The goal is
to reason about state, resources, trust, reproducibility, and cost as tools
change.
